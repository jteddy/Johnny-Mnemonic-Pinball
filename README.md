## EPROM Game Rom V1.2R

```
**Latest Version**
Version 1.2
Date: October 31, 1995
U6: Checksum: FD12 (R – Regular)
U6: Checksum: DB12 (B – Belgium/Canada)
```
**EPROM In My Machine on Purchase**

- V1.1R
- 27c040-12 / EP9236481

**What is this chip?**

4M, 120ns read time, 32 pin

**Other Chips to Consider**

- **Atmel AT27C040** - 70ns read time.
- 27C4001
- AT27C040-70PU
- W2E040
- W27C512

## NVRAM

For NVRAM I ordered

- RAMTRON FM1608 NVRAM (FM1608-120-PG) 
  - $13.30 for a 5 pack ($2.66 each inc delivery)
- 28 Pin socket



[PinSide - Installing Barakandl's NVRAM Battery Eliminator- Vid's Review](https://pinside.com/pinball/forum/topic/installing-barakandls-nvram-battery-eliminator-vids-review)
Andrew "Barakandl" here on pinside, sells RAMTRON FM1608 NVRAM chips that eliminate the need for batteries.
They cost (depending on how many you buy) $8-12 USD. This makes for a VERY affordable solution.

**Chips to Buy**

- RAMTRON FM1608 NVRAM
- Requires a 28 pin socet
- 1pc 6116 NVRAM Adapter - FM16W08 SMT to DIP24 - $15 + $2 shipping
- 16 adapter FM16W08 SMT to DIP24
- 6264 NVRAM Module
